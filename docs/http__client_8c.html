)

Make a HTTP POST request.

Warning
This function doesn't copy post_data, it's the responsibilty of user to keep it in scope.
Attention
Max URL size is set by HTTP_CLIENT_MAX_URL_LENGTH.
Note
When HTTP_CLIENT_POST_USE_STRLEN is passed as datalen, post_data length is checked used strlen.
Parameters
ipv4_addressDomain address (without http://).
pathPath after the domain (something like /docs/get_address.php).
post_dataData to add as post.
datalenPost data length.
Return values
ERR_C_NULL_POINTERIf ipv4_address, path or post_data is NULL.
ERR_C_OKon success.
