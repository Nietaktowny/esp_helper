◆ bmp_mode_t

enum bmp_mode_t
Enumerator
BMP_MODE_SLEEP 

Sensor does no measurements.

BMP_MODE_FORCED 

Sensor is in a forced measurement cycle. Sleeps after finishing.

BMP_MODE_NORMAL 

Sensor does measurements. Never sleeps.
